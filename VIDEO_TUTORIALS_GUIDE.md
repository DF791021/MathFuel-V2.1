## Tutorial 1: Creating Individual Certificates

**File:** `tutorial_01_individual_certificates.mp4`  
**Duration:** 2 minutes 30 seconds  
**Audience:** Teachers, Nutrition Staff  
**Difficulty Level:** Beginner

### What You'll Learn

This tutorial walks through the process of creating and printing a certificate for a single student. You'll learn how to navigate the Certificates tab, fill in student information, customize the achievement, and preview before printing.

### Key Steps Covered

1. Navigate to the Teacher Portal and click the Certificates tab
2. Select Individual Certificate
3. Enter the student's name and choose an achievement
4. Add an optional personalized message
5. Preview the certificate to ensure accuracy
6. Print or download as PDF

### When to Use This Tutorial

Use this tutorial when you need to recognize a single student's achievement, such as perfect quiz scores, excellent participation, or special accomplishments. This approach is ideal for immediate recognition or when you want to present a certificate to a student right away.

### Tips from the Tutorial

Always preview your certificate before printing to catch any spelling errors or formatting issues. The preview shows exactly how the certificate will appear on paper. You can customize the achievement text to match your specific recognition criteria.

## Tutorial 2: Bulk Certificate Generation and ZIP Download

**File:** `tutorial_02_bulk_certificates.mp4`  
**Duration:** 2 minutes 45 seconds  
**Audience:** Teachers, Nutrition Staff  
**Difficulty Level:** Beginner

### What You'll Learn

This tutorial demonstrates how to efficiently create certificates for an entire class at once. You'll learn how to enter multiple student names, preview all certificates together, and download them as a single ZIP file.

### Key Steps Covered

1. Navigate to Certificates and select Batch (Entire Class)
2. Enter student names one per line in the text area
3. Click Add Names to populate the student list
4. Review the student list and adjust as needed
5. Click Preview All to see thumbnail previews
6. Click Download All as ZIP to get all certificates in one file

### When to Use This Tutorial

Use this tutorial at the end of a unit, after a class challenge, or when recognizing all students for completing an activity. Bulk generation is much faster than creating individual certificates and is perfect for class-wide recognition events.

### Tips from the Tutorial

You can copy and paste student names from your class roster to save time. Always preview all certificates before downloading to ensure all names are spelled correctly. The ZIP file keeps all certificates organized in one place, making them easy to print or share.

## Tutorial 3: Customizing and Sending Emails with Certificates

**File:** `tutorial_03_email_certificates.mp4`  
**Duration:** 3 minutes 15 seconds  
**Audience:** Nutrition Staff, Teachers  
**Difficulty Level:** Intermediate

### What You'll Learn

This tutorial shows how to customize email subject lines and body text before sending certificates directly to your email address. You'll learn about variable substitution, email preview, and the sending process.

### Key Steps Covered

1. After adding student names, click Email ZIP
2. Customize the email subject line
3. Write a personalized email body message
4. Use variables like {student_count}, {school_name}, {teacher_name}, {date}
5. Click Preview Email to see the final result with variables substituted
6. Click Send ZIP to Email to send the certificates as an attachment

### When to Use This Tutorial

Use this tutorial when you want to email certificates to colleagues, parents, or administrators instead of printing them. The email customization feature lets you add context and personalization to your certificate distribution.

### Tips from the Tutorial

Variables automatically substitute with actual values when the email is sent. For example, {student_count} becomes "4" and {school_name} becomes "Lincoln Elementary School." This makes your emails personal and relevant without manual editing. Always preview your email to confirm that variables are substituted correctly and your message looks professional.

## Tutorial 4: Saving and Managing Email Templates

**File:** `tutorial_04_email_templates.mp4`  
**Duration:** 3 minutes  
**Audience:** Nutrition Staff, Teachers  
**Difficulty Level:** Intermediate

### What You'll Learn

This tutorial demonstrates how to save frequently-used email templates for quick reuse. You'll learn how to save, load, mark as default, and delete templates to streamline your workflow.

### Key Steps Covered

1. After customizing an email, enter a template name
2. Click Save to save the template
3. Open a new email dialog and find your saved template in the Email Templates section
4. Click Load to instantly populate the subject and body
5. Click the Star icon to mark a template as default
6. Click Delete to remove templates you no longer need

### When to Use This Tutorial

Use this tutorial to save time on repetitive email tasks. If you frequently send similar certificate emails with the same subject line and message structure, save it as a template. The next time you need to send certificates, load your template and make minor adjustments instead of starting from scratch.

### Tips from the Tutorial

Create templates for common scenarios, such as "Standard Class Certificate Email," "End of Unit Recognition," or "Special Achievement." Marking a template as default means it loads automatically the next time you open the email dialog, saving even more time. You can have multiple templates for different purposes and delete ones you no longer use.

## Troubleshooting Common Issues

### Issue: The Preview Shows Incorrect Information

**Solution:** Verify that all student names are spelled correctly in the student list. Check that you selected the correct students before previewing. If you notice an error, go back and correct the name in the list.

### Issue: Email Variables Are Not Substituting

**Solution:** Ensure you are using the correct variable names: {student_count}, {school_name}, {teacher_name}, or {date}. Variables are case-sensitive and must be enclosed in curly braces. Check the Email Templates section for a list of available variables.

### Issue: Template Did Not Save

**Solution:** Ensure you entered a template name before clicking Save. The template name field must not be empty. If you receive an error message, try again with a different template name.

### Issue: Cannot Load a Saved Template

**Solution:** Verify that you are in the correct email dialog (Email ZIP). Saved templates only appear when you click Email ZIP. If you still cannot find your template, try refreshing the page and opening the dialog again.
